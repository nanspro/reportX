A web app for users to interact with mythx API and generate beautiful HTML reports

## Run Locally

`export MYTHX_PASSOWRD='{Your Password}`

`export MYTHX_ETH_ADDRESS='{Your Address}`

`export FLASK_APP=gen.py`


To start the server
`python3 -m flask run`
